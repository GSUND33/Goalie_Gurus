The Model

We built a regression model based on the greatest goal scorers of all-time gamelogs. The model focuses on goals, shots, age, and game #. By combining these 4 inputs, we feed multiple machine learning models into each other in order to predict if and when Alex Ovechkin will catch Wayne Gretzky. The data is updated nightly with the latest gamelogs. Every prediction will be logged and timestamped on this page. Come back to see how accurate the model is!
